# Sortimo
Designs for bins and accessories to fit Sortimo T-BOXX cases.

Optimized for use with the OpenSCAD Customizer.

## Requirements

OpenSCAD-2025.01.06 or later.  Do NOT use the 2021.01 version.   https://openscad.org/downloads#snapshots

The BOSL2 Library for OpenSCAD.  https://github.com/BelfrySCAD/BOSL2
